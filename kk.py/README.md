# T-Shirt Sales Analysis Project

This Python project analyses T-shirt sales data from a CSV file and generates a business report.

## Project Files

- `tshirt_main.py` - Main program controller
- `tshirt_data_reader.py` - Reads CSV sales data
- `tshirt_analyzer.py` - Analyses sales by size, colour, and design
- `tshirt_report_writer.py` - Creates the final report
- `tshirt_sales.csv` - Sales dataset
- `tshirt_report.txt` - Generated report

## Features

- Reads sales data from a CSV file
- Calculates total revenue
- Calculates total units sold
- Calculates average price
- Finds top-performing sizes
- Finds top-performing colours
- Creates a text report

## How to Run

```powershell
py tshirt_main.py tshirt_sales.csv tshirt_report.txt
